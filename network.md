### 路由器的封包處理運作
1. MAC Addr;Media Access Contral Addr(physical Addr)  
   - 十六進制表示
   - 屬於硬體介面且獨一無二
   - 提供IEEE802系列區塊網技術(乙太網路、WiFi)的訊框(frame)接收方得知[來源位址]，或是在發送方設置[目的位址]
2. IP Addr(Logic Addr)
   - 32位元 0000.0000.0000.0000
3. port number： </br>
    80->http</br>
    443->https</br>
    22->ssh
4. host Addr
